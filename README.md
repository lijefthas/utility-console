# Utility Console

Utility Console is a desktop application for managing remote site controllers through a focused graphical interface.

It brings common support and maintenance workflows into one place, including connection profiles, health checks, log review, service control, database inspection, module maintenance, and developer support tools.

The application is intended for controlled technical environments where operators need a faster, more consistent way to inspect devices, diagnose issues, and perform routine maintenance without manually switching between multiple tools.

## Overview

- Manage saved connection profiles for frequently accessed devices.
- Inspect system state, logs, services, and module details from a single UI.
- Follow guided maintenance workflows with progress feedback and rollback support.
- Review database information through an integrated query interface.
- Use developer-focused tools for project lookup, build assistance, and environment setup.
